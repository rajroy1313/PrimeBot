Getting Started

Welcome to PrimeBot! Follow these simple steps to get started:

1. Invite PrimeBot to Your Server

Click the invite link and select your server. Make sure you have "Manage Server" permissions.

🔗 Invite PrimeBot to Your Server

2. Set Up Permissions

PrimeBot requires the following permissions to function properly:

  • Send Messages
  • Embed Links
  • Attach Files
  • Add Reactions
  • Manage Messages
  • Manage Channels (for tickets)
  • Manage Roles (for leveling)

3. First Commands

Try these commands to get familiar with PrimeBot:

/help - View all available commands
/about - Learn more about PrimeBot
/stats - View bot statistics

Command Reference

PrimeBot supports both slash commands (/) and prefix commands ($). Here's a complete list:

General Commands

/help - Show command list
/about - Bot information
/stats - Bot statistics
/echo [text] - Repeat a message

Giveaway Commands

/giveaway - Create a new giveaway
/end [message_id] - End a giveaway
/reroll [message_id] - Reroll winners

Leveling Commands

/leveling - Manage leveling system
/stats - View your level and XP

Ticket Commands

/ticket - Manage ticket system
/createticket - Create a new ticket
/tickethistory - View ticket history

Giveaway System

Create engaging giveaways to boost server activity and reward your community members.

Creating a Giveaway

Use the /giveaway command to create a new giveaway:

/giveaway duration:1d winners:1 prize:Discord Nitro

Duration Formats

  • s - Seconds (e.g., 30s)
  • m - Minutes (e.g., 15m)
  • h - Hours (e.g., 2h)
  • d - Days (e.g., 7d)

Managing Giveaways

You can end giveaways early or reroll winners:

/end message_id:123456789 - End a giveaway early
/reroll message_id:123456789 - Pick new winners

Features

  • Automatic winner selection
  • Customizable duration and prizes
  • Multiple winners support
  • Reroll functionality
  • Automatic notifications

Leveling System

Reward active members with an engaging leveling system that tracks activity and awards XP.

How It Works

  • Members earn XP by sending messages
  • XP gain has a cooldown to prevent spam
  • Level up announcements in designated channels
  • Role rewards for reaching certain levels

Configuration

Set up the leveling system using:

/leveling setup - Initial setup
/leveling channel - Set announcement channel
/leveling roles - Configure level roles

XP System

  • XP per message: 15-25 XP (random)
  • Cooldown: 60 seconds
  • Level formula: XP required = level × 100

Troubleshooting

Common issues and solutions for PrimeBot usage.

Bot Not Responding

  • Check bot permissions in the channel
  • Ensure bot has "Send Messages" permission
  • Try using slash commands instead of prefix commands
  • Check if bot is online in member list

Commands Not Working

  • Use /help to see available commands
  • Check command syntax and parameters
  • Ensure you have required permissions
  • Try re-inviting the bot with updated permissions

Giveaways Issues

  • Ensure bot can add reactions to messages
  • Check that bot has "Embed Links" permission
  • Verify message ID format for end/reroll commands

Leveling Not Working

  • Check if leveling is enabled for your server
  • Ensure bot can read message history
  • Verify XP gain cooldown (60 seconds)
  • Check channel permissions
